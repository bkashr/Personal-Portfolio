Project Overview

This project represents a comprehensive analysis of NHL betting markets, specifically focusing on "goal scored in first 5/10 minutes" prop bets. After identifying potential opportunities in FanDuel's niche betting options, I systematically collected real-world data across 32 games, tracking team matchups, over/under odds, goal timing statistics, and betting outcomes.

The project evolved from simple data collection into a sophisticated martingale simulation framework that evaluates betting strategies across thousands of iterations. This data-driven approach bridges sports analytics with algorithmic trading concepts, demonstrating the practical application of statistical modeling in real-world scenarios.

Martingale Simulation Framework

The core of this project lies in the martingale simulation algorithm, which tests betting strategies under various scenarios. The simulation framework allows for comprehensive analysis of risk management, bankroll optimization, and strategy validation.

Simulation Parameters

  • Initial Bankroll: $150
  • Initial Bet: $2.50
  • Max Iterations: 100
  • Bet Doubling: On losses (classic martingale)
  • Stop Conditions: Bankroll depletion or insufficient funds
# Martingale Simulation Logic
def simulate_martingale(success_rate, odds, initial_bankroll, initial_bet, max_iterations):
    bankroll = initial_bankroll
    current_bet = initial_bet
    
    for iteration in range(max_iterations):
        if bankroll < current_bet:
            break
            
        # Simulate bet outcome
        if random.random() < success_rate:
            # Win: increase bankroll, reset bet
            bankroll += current_bet * (odds - 1)
            current_bet = initial_bet
        else:
            # Loss: decrease bankroll, double bet
            bankroll -= current_bet
            current_bet *= 2
    
    return bankroll

Analysis Results

The simulation results reveal critical insights about the viability of different betting strategies. The analysis demonstrates the inherent risks and opportunities present in each approach, providing a realistic assessment of potential outcomes.

Strategy Comparison

5-Minute Goal Strategy: While offering higher potential returns due to favorable odds, this strategy suffers from high volatility and frequent losing streaks. The 34.38% success rate creates significant risk of rapid bankroll depletion.

10-Minute Goal Strategy: The more conservative approach with 68.75% success rate provides greater stability and reduced risk of total loss. However, the lower odds (-147 American) result in smaller individual profits.

Realistic Outcomes: Simulations consistently show bankroll depletion scenarios, highlighting that martingale strategies, while mathematically sound in theory, face practical limitations in real-world betting environments.

class HockeyBettingAnalyzer:

        self.process_odds()
        self.calculate_success_rates()
    
    def process_odds(self):
        # Convert American odds to decimal
        self.data['5min_decimal'] = self.data['5 Minute Goal Odds'].apply(

        )
