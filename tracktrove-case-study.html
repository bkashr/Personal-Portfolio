TRACKTROVE

Revolutionizing Music Discovery Through Community & Innovation
Timeline 2023
Role UX Researcher/UI Designer
Tools Figma, User Research, Product Strategy
01

PROJECT OVERVIEW

The Challenge

Current music streaming platforms like Spotify and Apple Music excel at music delivery but fall short in creating meaningful social connections and community engagement. Users lack ways to track their musical journey, share discoveries, or connect with fellow music enthusiasts beyond basic playlist sharing.

The Solution

TrackTrove bridges the gap between music streaming and social interaction by creating a community-driven platform that seamlessly integrates with existing streaming services. Users can track listening history, comment on songs, discover new music through personalized recommendations, and build connections with other music lovers.

Key Features

  • Comprehensive Tracking: Monitor listening history, create personal music timelines
  • Social Interaction: Comment on songs, rate albums, share musical discoveries
  • Personalized Discovery: AI-powered recommendations based on listening patterns
  • Multi-Platform Integration: Seamless connection with Spotify, Apple Music, and more
  • Community Building: Connect with artists, producers, and fellow music enthusiasts
02

USER TYPES & ROLES

Listeners

Teenagers and adults who enjoy using music applications for creative expression and discovery.

Key Actions:

  • Discover and listen to music
  • Customize profiles
  • Private messaging with friends
  • Comment and rate music
  • Create private/public accounts

Artists

Local and well-known artists looking to showcase music and engage with their fanbase.

Key Actions:

  • Showcase music and albums
  • Sell event tickets and merchandise
  • Engage with fans directly
  • Announce important updates
  • Build fan communities

Music Producers

Producers worldwide managing artists and songs while building their professional network.

Key Actions:

  • Upload and share music
  • Manage artist profiles
  • Announce releases and events
  • Sell merchandise
  • Collaborate with artists

Record Labels

Private and well-known labels managing artist portfolios and releasing music.

Key Actions:

  • Manage artist portfolios
  • Release and promote music
  • Access data analytics
  • Collaborate with artists
  • Sell merchandise

Radio Stations

Local favorites and well-known stations promoting events and engaging listeners.

Key Actions:

  • Edit and share playlists
  • Promote events and shows
  • Live stream content
  • Host audience polls
  • Spotlight emerging artists

METADATA SCHEMA DESIGN

Schema Architecture

The TrackTrove metadata schema is designed to handle complex relationships between users, music, and social interactions while maintaining flexibility for future expansion.

Song Metadata

Property Type Required Example
Name String Yes "Good 4 U"
Artist String Yes "Olivia Rodrigo"
Genre Controlled Vocabulary Yes Pop, Rock, EDM, etc.
Duration Time Yes 2:58
Streams Integer Yes 1,998,679,907

User Metadata

Property Type Required Example
Username String Yes "@mangopomelo"
Verification Boolean + Icon No ☑ Verified
Favorite Genre Array No ["Pop", "Hip Hop"]
Join Date Date Yes Feb 3, 2023
Followers Integer Yes 500

DESIGN PROCESS

Conducted comprehensive analysis of existing music platforms, identified user pain points, and researched competitor features to understand market gaps.

Defined six distinct user types with detailed personas, use cases, and feature requirements to ensure comprehensive platform coverage.

03

Designed flexible metadata schema to handle complex relationships between users, music, and social interactions while maintaining scalability.

04

Created detailed UI prototypes for key user flows, focusing on intuitive navigation and seamless integration with existing streaming services.

05

Conducted user testing with target audiences, gathered feedback, and iterated on designs to ensure optimal user experience across all user types.
